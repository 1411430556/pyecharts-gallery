## 项目简介

* 项目基于 Pycharts 1.0.0 版本进行展示
* 百度官方 Echarts [官方实例](https://echarts.baidu.com/examples/)

## 项目须知

* 项目代码结构按照 Echarts 官网的实例模块进行划分
* 代码内有根据 1.0.0 版本的 Pyecharts 所生成的可视化数据视图和官方的进行对比, 有列出能实现的功能以及未实现的功能
